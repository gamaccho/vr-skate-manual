🤖 VS Code で Claude Code を
使えるようにしよう

このマニュアルでは、VS Code上でAIコーディングアシスタント「Claude Code」を使えるようにするまでの手順を案内します。

💡 CLIメインの運用について Robloxスクリプティングのような本格的な作業では、VS CodeのターミナルからCLIでClaudeを呼び出すのがメインの使い方です。プロジェクトフォルダに cd して claude コマンドを実行すると、そのディレクトリ全体をClaudeが把握した状態で作業できます。

エディター右上のスパークアイコン（✦）からの操作は、ファイル単体への軽微な修正など限定的な用途向けです。
⚠️ 事前に確認 Claude Codeの使用にはClaudeの有料プラン（Pro以上）が必要です。無料プランでは利用できません。

まず、Claudeのアカウントはすでに持っていますか？

ステップ 1-a

🔐 Claudeアカウントを作成する

  • 1
    ブラウザで https://claude.ai を開く
  • 2
    右上または中央の Sign up をクリック
  • 3
    メールアドレスで登録、またはGoogleアカウントで続行
  • 4
    確認メールが届いたらリンクをクリックして認証を完了する
ステップ 2

💳 有料プランに加入していますか？

Claude Codeを使うにはProプラン以上の加入が必要です。

プラン月額Claude Code
Free（無料）$0❌ 使用不可
Pro ★おすすめ$20 / 月✅ 使用可
Max$100〜$200 / 月✅ 使用量大
💡 どのプランを選ぶ？ 個人での開発・学習目的であればPro（$20/月）で十分です。

有料プランに加入していますか？

ステップ 2-a

💳 Proプランにアップグレードする

  • 1
    https://claude.ai にログインする
  • 2
    左下のアカウントメニュー → Upgrade plan をクリック
  • 3
    Pro プランを選択して Subscribe をクリック
  • 4
    クレジットカード情報を入力して支払いを完了する（$20/月）
⚠️ 注意 支払いはAnthropicに対して行われます。サブスクリプションはいつでもキャンセル可能です。
ステップ 3

🖊️ VS Codeのバージョンを確認する

Claude Codeの拡張機能にはVS Code 1.98.0 以上が必要です。

  • 1
    Mac： メニューバー → Code → About Visual Studio Code
    Windows： メニューバー → Help → About
  • 2
    バージョン番号が1.98.0 以上であれば問題なし
ステップ 3-a

⬆️ VS Code をアップデートする

  • 1
    Mac / Windows： メニューバー → Help → Check for Updates
  • 2
    Download Update → Restart to Apply
  • 3
    再起動後、バージョンが 1.98.0 以上になっていることを確認する
ℹ️ 手動でダウンロードする場合https://code.visualstudio.com/download から最新版を入手できます。
ステップ 4

🔌 Claude Code拡張機能をインストールする

  • 1
    左サイドバーの拡張機能アイコン（田）をクリック
    またはショートカット：Cmd+Shift+X（Mac）／ Ctrl+Shift+X（Windows）
  • 2
    検索バーに Claude Code と入力する
  • 3
    発行者が Anthropic のものを選ぶ（anthropic.claude-code）
    ⚠️ 注意Anthropic公式以外の「Claude Code」という名前の拡張機能も存在します。必ず発行者がAnthropicのものをインストールしてください。
  • 4
    Install（インストール）をクリック
  • 5
    インストール後にアイコンが表示されない場合はVS Codeを再起動する
    Cmd+Shift+P → Developer: Reload Window でも可
ステップ 5

🔑 Claudeアカウントでサインインする

  • 1
    VS Codeエディター右上に✦（スパークアイコン）が表示される
    ※ファイルを開いていないと表示されません
  • 2
    スパークアイコンをクリックすると、右側にClaudeのパネルが開く
  • 3
    パネル内の Sign in ボタンをクリックする
  • 4
    ブラウザが開くので、Claudeアカウントでログインして認証を完了する
  • 5
    VS Codeに戻ると、Claudeのチャットパネルが使えるようになっている 🎉
💡 スパークアイコンが見当たらない場合VS Code下部ステータスバー右端の✱ Claude Codeをクリックしても開けます。またはコマンドパレット（Cmd+Shift+P）から Claude Code と入力して選択。
ステップ 6

⌨️ CLIメインの使い方

本格的な作業はVS Codeのターミナルから claude コマンドで起動するのがメインの運用です。

① プロジェクトフォルダをVS Codeで開く

  • 1
    VS Codeのメニュー → File → Open Folder でプロジェクトフォルダを開く
    またはターミナルからフォルダに移動して code . を実行
  • 2
    左サイドバーのエクスプローラー（📄アイコン）にファイル一覧が表示されることを確認する

② VS Codeのターミナルを開く

  • 1
    メニュー → Terminal → New Terminal
    またはショートカット：Ctrl+`（Mac: Cmd+`）
  • 2
    ターミナルがVS Code下部に開き、カレントディレクトリがプロジェクトフォルダになっていることを確認する

③ プロジェクトフォルダに移動する

cd ~/Desktop/プロジェクトフォルダ名
💡 現在地の確認pwd と入力するとカレントディレクトリのパスが確認できます。

④ claude コマンドを実行する

claude
  • 1
    画面に従い、Enterとか押しながら進む。有料課金していれ ❯ 1. Claude account with subscription を選択

    ▼ claudeコマンド起動直後の画面

  • 2
    ブラウザが開くのでClaudeアカウントでログインして認証を完了する

    ▼ ブラウザのログイン画面

  • 3
    左サイドバーのエクスプローラーにファイル一覧、下部ターミナルにClaudeのプロンプトが表示された状態で作業できる 🎉

    ▼ ログイン完了後の状態

😊 画面の指示になんとなく従っていけば大丈夫です！

▼ VS Codeでのワークスペースのレイアウト例

⑤ 基本的な操作

使い方の例プロンプトに日本語で指示するだけでOKです：

「ServerScriptService/GameManager.lua のエラーを直して」
「敵キャラのAIスクリプトを追加したい」
「このフォルダのスクリプト構成を説明して」
💡 終了するには/exit または Ctrl+C

CLIとスパークアイコンの使い分け

操作方法向いている場面
⌨️ CLI（ターミナル）スクリプト全体の修正・機能追加・複数ファイルにまたがる本格的な開発
✦ スパークアイコン開いているファイルの一部をさっと直したいなど軽微な修正
⚠️ 使用量の上限についてProプランには使用量の制限があります。上限に達した場合は時間をおいて再試行してください。
