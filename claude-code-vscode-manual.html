🤖 VS Code で Claude Code を
使えるようにしよう

このマニュアルでは、VS Code上でAIコーディングアシスタント「Claude Code」を使えるようにするまでの手順を案内します。

⚠️ 事前に確認 Claude Codeの使用には Claudeの有料プラン（Pro以上） が必要です。無料プランでは利用できません。詳しくはこのマニュアルの中で説明します。

まず、Claudeのアカウントはすでに持っていますか？

ステップ 1-a

🔐 Claudeアカウントを作成する

  • 1
    ブラウザで https://claude.ai を開く
  • 2
    右上または中央の Sign up をクリック
  • 3
    メールアドレスで登録、またはGoogleアカウントで続行
  • 4
    確認メールが届いたらリンクをクリックして認証を完了する
  • 5
    ログインできたらアカウント作成完了。次のステップで課金プランを確認します
ステップ 2

💳 有料プランに加入していますか？

Claude Codeを使うには Proプラン以上 の加入が必要です。

プラン月額Claude Code
Free（無料） $0 ❌ 使用不可
Pro ★おすすめ $20 / 月 ✅ 使用可
Max $100〜$200 / 月 ✅ 使用量大
💡 どのプランを選ぶ？ 個人での開発・学習目的であれば Pro（$20/月） で十分です。使用量が多くなってきたら Maxプランへの移行を検討してください。

有料プランに加入していますか？

ステップ 2-a

💳 Proプランにアップグレードする

  • 1
    https://claude.ai にログインする
  • 2
    左下のアカウントメニュー → Upgrade plan をクリック
  • 3
    Pro プランを選択して Subscribe をクリック
  • 4
    クレジットカード情報を入力して支払いを完了する（$20/月）
⚠️ 注意 支払いはAnthropicに対して行われます。サブスクリプションはいつでもキャンセル可能です。
ステップ 3

🖊️ VS Codeのバージョンを確認する

Claude Codeの拡張機能には VS Code 1.98.0 以上 が必要です。

バージョン確認方法

  • 1
    VS Codeを開く
    Mac： メニューバー → Code → About Visual Studio Code
    Windows： メニューバー → Help → About
  • 2
    表示されたバージョン番号が 1.98.0 以上 であれば問題なし

バージョンは1.98.0以上でしたか？

ステップ 3-a

⬆️ VS Code をアップデートする

  • 1
    Mac： メニューバー → Help → Check for Updates
    Windows： メニューバー → Help → Check for Updates
  • 2
    アップデートがあれば Download Update → Restart to Apply
  • 3
    再起動後、バージョンが 1.98.0 以上になっていることを確認する
ℹ️ アップデートが表示されない場合 https://code.visualstudio.com/download から最新版を直接ダウンロードできます。
ステップ 4

🔌 Claude Code拡張機能をインストールする

  • 1
    VS Codeの左サイドバーにある 拡張機能アイコン（田） をクリック
    またはキーボードショートカット：Cmd+Shift+X（Mac）／ Ctrl+Shift+X（Windows）
  • 2
    検索バーに Claude Code と入力する
  • 3
    発行者が Anthropic のものを選ぶ（anthropic.claude-code）
    ⚠️ 注意 Anthropic公式以外の「Claude Code」という名前の拡張機能も存在します。必ず発行者が Anthropic のものをインストールしてください。
  • 4
    Install（インストール） をクリック
  • 5
    インストール後にアイコンが表示されない場合は、VS Codeを再起動する
    Cmd+Shift+P → Developer: Reload Window でも可
🖼️
拡張機能の検索画面（スクリーンショットをここに追加できます）
ステップ 5

🔑 Claudeアカウントでサインインする

  • 1
    VS Codeエディター右上に ✦（スパークアイコン） が表示される
    ※ファイルを開いていないと表示されません。適当なファイルを開いてください
  • 2
    スパークアイコンをクリックすると、右側にClaudeのパネルが開く
    🖼️
    スパークアイコンの位置（スクリーンショットをここに追加できます）
  • 3
    パネル内の Sign in ボタンをクリックする
  • 4
    ブラウザが開くので、Claudeアカウントでログインして認証を完了する
  • 5
    VS Codeに戻ると、Claudeのチャットパネルが使えるようになっている 🎉
💡 スパークアイコンが見当たらない場合 VS Code下部のステータスバー右端に ✱ Claude Code と表示されているのでそちらをクリックしても開けます。またはコマンドパレット（Cmd+Shift+P）から Claude Code と入力して選択。
ステップ 6

💬 Claude Codeの基本的な使い方

① コードについて質問する

チャットパネルの入力欄に自由に質問を入力してEnterを押すだけです。

例 「このスクリプトのエラーの原因を教えて」
「◯◯という機能を追加するにはどうすればいい？」
「このコードをもっと読みやすく書き直して」

② ファイルを指定して質問する（@メンション）

入力欄で @ と入力するとファイル名の候補が出ます。ファイルを選ぶとClaudeがそのファイルを読んだ状態で返答してくれます。

💡 ショートカット エディターでコードを選択した状態で Option+K（Mac）／Alt+K（Windows）を押すと、そのファイルと行番号を自動でメンションできます。

③ 変更の確認と承認

Claudeがファイルを編集する場合、変更前と変更後の差分が表示されます。内容を確認してから Accept（承認） または Reject（却下） を選べます。

🖼️
差分表示画面（スクリーンショットをここに追加できます）

④ パネルの場所を変える

Claudeのパネルはドラッグして好きな位置に移動できます。右サイドバーに置くとコードを書きながら使いやすいです。

⚠️ 使用量の上限について Proプランには1日あたりの使用量制限があります。上限に達した場合は翌日にリセットされます。
